## 📖 Overview

This project is a **single-file interactive HTML roadmap** designed to help anyone — from beginners to experienced professionals — navigate the cybersecurity landscape. Each domain card includes:

- ✅ **Core topics & subtopics** with key tools and platforms
- 📚 **Certifications** to target per domain
- 🆓 **Free learning resources** — YouTube, labs, docs, CTFs, and free courses

No paywalls. No fluff. Just a clear, structured path.

## 🚀 Getting Started

### View the Roadmap

Just open the HTML file in any modern browser — no server, no dependencies, no installation required.

```bash
git clone https://github.com/your-username/cybersecurity-roadmap.git
cd cybersecurity-roadmap
open cybersecurity-roadmap-with-resources.html   # macOS
# or
xdg-open cybersecurity-roadmap-with-resources.html  # Linux
# or simply double-click the file in Windows Explorer
```

### What You'll See

- A **skill progression bar** at the top (Foundational → Intermediate → Advanced)
- **12 interactive domain cards**, each with two tabs:
  - `Topics` — core subtopics, tools, and target certifications
  - `Free Resources` — curated free YouTube, labs, docs, courses, and CTFs
- A **global free platforms strip** at the top covering cross-domain resources
